My Business - invoice app - about

Main features:

  • user authentication
  • create, update, delete invoices
  • manage company profile
  • manage product and contractor profiles
  • create and download PDF invoice

Getting Started and installing

  • $ git clone https://github.com/michal-siedlecki/my_business
  • $ source venv/bin/activate
  • $ pip install -U -r requirements.txt
  • $ python manage.py runserver
  • open browser at localhost (127.0.0.1:8000) and feel free to test

Prerequisites

What things you need to install the software:
  • Python 3+
  • Django 3.0.4
  • django-weasyprint
  • django-crispy-forms

Built With
